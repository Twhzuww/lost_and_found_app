# Lost & Found Mobile Application

A Flutter-based Lost & Found application that allows users to:
- Register and login using Firebase Authentication
- Report lost or found items with GPS location
- View nearby items and hazards on Google Maps
- Track user location in real time
- Manage profile and change password
